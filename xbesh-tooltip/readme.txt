Displays a "Built with xBesh" tooltip at the bottom of every page on your WordPress site.

== Description ==

xBesh Tooltip is a simple plugin that adds a "Built with xBesh" tooltip to the bottom of every page on your WordPress site. The tooltip can be positioned on the left, center, or right side of the page.

Features:

* Lightweight and easy to use
* Customizable position (left, center, right)
* Enable/disable from the admin settings
* Responsive design

== Installation ==

1. Upload the `xbesh-tooltip` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin settings under 'Settings > xBesh Tooltip'

== Frequently Asked Questions ==

= Can I customize the tooltip text? =

Currently, the tooltip text is fixed as "Built with xBesh". Future versions may include customization options.

= Does this plugin slow down my website? =

No, this plugin is very lightweight and has minimal impact on your website's performance.
